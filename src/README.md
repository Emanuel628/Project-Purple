# src

This folder will contain the Project Purple application source code when development begins.

Phase 1 will start with the static app shell. Backend logic, authentication, video upload, and streaming infrastructure should not be added here until the proper build phase.

Planned future organization:

```txt
src/
  app/
  components/
  data/
  pages/
  styles/
  utils/
```

Keep source code organized by purpose. Avoid dumping unrelated files directly into this folder.
