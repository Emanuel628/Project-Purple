# public/js

JavaScript for the Project Purple static app shell.

Planned organization:

```txt
public/js/
  app.js
  theme.js
  mock-data.js
  ui.js
  pages/
```

Rules:

- Keep Phase 1 JavaScript minimal.
- Do not add backend calls during the static shell phase.
- Use `mock-data.js` for local demo content in Phase 2.
- Use `theme.js` for light/dark/system appearance handling.
- Use `ui.js` for shared UI behavior.
